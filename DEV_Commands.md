# DEV Commands — Phantombite Creatures

Alle MES Commands erfordern Admin-Rechte.
Singleplayer: immer erlaubt. Multiplayer: Admin erforderlich.

## MES Spawn Commands

```
/MES.Spawn.Creature [SubtypeId]
```

| Command | Beschreibung |
|---------|-------------|
| `/MES.Spawn.Creature Shark` | Shark manuell spawnen |
| `/MES.Spawn.Creature Groump` | Groump manuell spawnen |
| `/MES.Spawn.Creature Raptor` | Raptor manuell spawnen |
| `/MES.Spawn.Creature OrganicPlasma` | OrganicPlasma manuell spawnen |
| `/MES.Spawn.Creature Space_spider_worm` | Worm manuell spawnen |
| `/MES.Spawn.Creature Space_spider_hugeworm` | HugeWorm manuell spawnen |

## MES Einstellungen

| Command | Beschreibung |
|---------|-------------|
| `/MES.Settings.Creatures.EnableSpawns.true` | Creature Spawns aktivieren |
| `/MES.Settings.Creatures.EnableSpawns.false` | Creature Spawns deaktivieren |
| `/MES.Settings.Creatures.OverrideVanillaCreatureSpawns.true` | MES übernimmt kompletten Creature Spawn |
| `/MES.Settings.Creatures.OverrideVanillaCreatureSpawns.false` | Vanilla Spawn wieder aktiv |

## Hinweise
- `OverrideVanillaCreatureSpawns` ersetzt den Planet Creature Spawner Mod vollständig.
- Spawn Commands spawnen die Kreatur direkt beim Spieler.
- Tag/Nacht Einschränkungen gelten nicht für manuelle Spawn Commands.
